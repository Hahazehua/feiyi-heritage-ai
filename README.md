# HeritageLink AI｜非遗礼遇

HeritageLink AI 是一个面向非遗礼品推荐与定制沟通的可解释 MVP。当前使用飞颐铁画的 8 件演示产品，验证从客户礼赠需求、规则推荐、双语文化介绍到商家定制需求单的完整交易前流程。

> 当前产品、价格、产能、材料、交期和运输信息仅用于MVP演示，不代表飞颐铁画真实商业承诺。

本项目属于 SynNovator 数字文化赛道（原赛道标识：`track-98`）。

## 用户流程

1. 用户填写客户类型、赠礼对象、单件预算、数量、场景、风格、文化寓意、定制要求、目的地和交付天数。
2. 系统先执行预算、数量、交期、定制、Logo 和国际运输硬性过滤。
3. 合格产品按固定的 100 分规则评分，最多展示 3 件，并逐项解释得分和风险。
4. 每件产品展示基于本地资料模板生成的中英文文化介绍，不补写未知事实。
5. 用户选择产品后生成 JSON 定制需求单，包含预算、数量、主题、题字、Logo、包装、目的地、交期和商家待确认问题。
6. 如果没有合格产品，系统说明冲突原因并给出调整建议，不强行推荐。

## 技术范围

- Python 3.11+
- Streamlit
- pandas
- 本地 CSV/JSON
- pytest 与 Ruff
- 确定性规则评分和模板文案

当前不接入外部大模型 API，不使用 RAG、向量数据库、数据库或 ORM。

## 安装

建议在独立虚拟环境中运行：

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -e .[dev]
```

## 启动演示

```powershell
python -m streamlit run app.py
```

浏览器通常会自动打开 `http://localhost:8501`。如果端口被占用，可增加 `--server.port 8502`。

## 测试与代码检查

```powershell
python -m ruff format --check .
python -m ruff check .
python -m pytest
```

无界面冒烟启动：

```powershell
python -m streamlit run app.py --server.headless true
```

## 演示数据

`data/demo/` 当前包含：

- 1 个飞颐铁画演示商家；
- 1 个官方级别标记为 `unverified` 的铁画演示项目；
- 8 件铁画演示产品；
- 每件产品各一条中文和英文演示文案；
- 题字、Logo、包装、图案等演示定制能力。

所有相关记录均标注 `is_demo=true`。仓库不声明真实传承人身份、官方认证级别、政府背书、真实价格、产能、材料、交期或运输能力。

## 当前限制

- 只有一个试点商家和少量演示产品；
- 推荐依赖人工维护的标签与固定权重，不是个性化学习模型；
- 双语内容来自本地字段和模板，全部仍需商家审核；
- 国际运输支持只是演示过滤字段，不代表真实可达性、运费或合规承诺；
- 不提供登录、支付、库存、合同、物流、结算或商家后台；
- 不保存客户个人身份和联系方式。

## 后续 AI 与 RAG 计划

在规则基线经过商家和真实用户验证后，可逐步增加大模型能力，用于理解更自然的需求、辅助整理商家资料、生成待审核双语草稿和改写沟通文本。之后可评估使用 RAG 检索经过授权和审核的非遗资料。

未来增强必须继续保留当前硬性约束过滤、规则基线、来源标注、商家审核和可追溯需求快照。大模型不会替代价格、产能、交期、运输和文化事实的人工确认。
